官方 CSV 同步到 SQLite，并用 Streamlit 做图表与表格。本页为部署在 Vercel 上的静态说明；交互式应用请用下方方式运行。

为何不在此站打开 Streamlit？

Streamlit 是长连接 Web 应用，依赖本地或持久化 SQLite。 Vercel Serverless 不适合托管标准 Streamlit 进程（超时、无持久磁盘、架构不匹配）。

推荐：Streamlit Community Cloud

连接 GitLab 仓库，云端运行 streamlit_app.py：

  • 在 share.streamlit.io 用 GitLab 登录并导入本仓库。
  • Main file 填 streamlit_app.py，Python 版本与依赖按仓库配置。
  • 数据库需随部署提供（例如构建时下载、或只读外链），详见仓库 README。

本地运行

poetry install → poetry run python sync_c4.py ... → poetry run streamlit run streamlit_app.py
